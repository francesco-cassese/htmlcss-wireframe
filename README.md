# Repository: htmlcss-wireframe

## Obiettivo dell'Esercizio
Riprodurre il layout del wireframe assegnato focalizzandosi sulle proprietà del **Box Model** (`margin`, `padding`, `border`).

## Note di sviluppo
- Priorità alla **coerenza visiva** e all'armonia degli spazi.
- Layout strutturato per ottenere distanze proporzionate, come richiesto dalla consegna.

## Tecnologie
- HTML5
- CSS3

### Struttura delle cartelle
```text
htmlcss-wireframe/
├── css/
│   └── style.css       # Foglio di stile con logica Box Model
├── index.html          # Markup strutturale
└── README.md           # Documentazione del progetto
```
